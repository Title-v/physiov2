`/` is now a native Next Therapist home page. `/therapist`,
`/therapist/capture`, `/therapist/plan`, `/therapist/record`, and
`/therapist/dashboard` are native Next pages.

Legacy `.html` URLs still rewrite into their native routes, including
`/therapist/capture.html`, `/therapist/plan.html`, `/therapist/record.html`,
and `/therapist/dashboard.html`. Keep Therapist capture, plan builder, patient
roster, built-in exercises, boundary boxes, angle overlays, motion scoring, and
trajectory references working from these native route locations.
