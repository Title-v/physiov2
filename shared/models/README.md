# PhysioAI Shared Models

Browser-loadable model artifacts live here.

- `motion-tcn/manifest.json` describes the active assistive motion classifier.
- `motion-tcn/model.json` and weight shards are generated by `npm run train:tcn` or prepared by `npm run convert:tcn`.

These files are served through `/shared/models/*` by the app's public symlink. Keep trained model artifacts out of the frontend runtime bundle; they should load lazily and remain optional.
